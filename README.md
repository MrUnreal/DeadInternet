# Dead Internet

Interactive visualization of the Dead Internet phenomenon — watch the web transform from human to bot-dominated.

Live at [mrunreal.github.io/DeadInternet](https://mrunreal.github.io/DeadInternet)

## Features
- Timeline slider (2010–2026) with auto-play
- 220-particle network visualization
- Simulated social media feed
- Real data from Cloudflare & Imperva research
- Glitch effects that intensify with bot traffic
